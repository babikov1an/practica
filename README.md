# Детекция проезда на красный свет

AI-система для детекции автомобилей, проезжающих на красный свет.

## Установка

1. Установите зависимости:
```bash
pip install -r red_light_detector/requirements.txt
```

2. Запустите приложение:
```bash
python red_light_detector/app.py
```

3. Откройте браузер: http://localhost:5001

## Функционал

- Загрузка изображений перекрестков
- Детекция автомобилей и светофоров с помощью YOLOv8
- Определение цвета сигнала светофора (красный/зеленый/желтый)
- Автоматическое определение нарушений (проезд на красный)
- Экспорт отчетов в PDF и Excel
- История обработки изображений

## Структура проекта

```
red_light_detector/
├── app.py              # Flask-сервер
├── detector.py         # Логика детекции
├── requirements.txt    # Зависимости
├── static/
│   ├── css/style.css   # Стили
│   ├── js/main.js      # JavaScript
│   ├── uploads/        # Загруженные изображения
│   └── results/        # Обработанные изображения
├── templates/
│   └── index.html      # Веб-интерфейс
└── reports/            # Сгенерированные отчеты
```

## Технологии

- Python 3.13
- Flask - веб-фреймворк
- YOLOv8 - детекция объектов
- OpenCV - обработка изображений
- SQLite - хранение истории
- ReportLab - генерация PDF
- OpenPyXL - генерация Excel

## Использование

1. Загрузите изображение перекрестка
2. Нажмите "Запустить обработку"
3. Просмотрите результаты: количество автомобилей, светофоров, нарушений
4. Скачайте отчет в PDF или Excel
